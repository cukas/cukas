# Nicolas Cukas

I like building tools that make me faster. Mostly plugins, compilers, and things that remove friction from how I work.

## What I'm into

I build a lot of plugins for Claude Code — stuff like [remembrall](https://github.com/cukas/remembrall) (auto-saves context so I never lose work), [claudes-ai-buddies](https://github.com/cukas/claudes-ai-buddies) (call Codex and Gemini as peer reviewers from one terminal), [patrol](https://github.com/cukas/patrol) (ESLint policy engine with zero-token overhead), and [hello-claude](https://github.com/cukas/hello-claude) (lets multiple Claude instances talk to each other). I build them because they solve problems I actually hit every day, and they keep me moving.

I also built [KERN](https://github.com/KERNlang/kern-lang) — a programming language designed for LLMs. The cool part: it's a 192-line spec that compiles to 12 different framework targets (Next.js, React, Vue, FastAPI, React Native, etc.), and LLMs don't need to learn it — they just get it. [kernlang.dev](https://kernlang.dev) is fully compiled from .kern source files. There's also a [security scanner for MCP servers](https://open-vsx.org/extension/KERNlang/kern-mcp-security) that came out of it.
